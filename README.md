# ⚔️ Yu-Gi-Oh! Forbidden Memories Fusions

An interactive web application to explore fusion combinations from the classic PlayStation game **Yu-Gi-Oh! Forbidden Memories**. Browse cards, view their stats, and discover all possible fusion recipes to create powerful monsters!

## 🎮 Features

- **Interactive Card Gallery**: Browse through all fusion cards with beautiful card images
- **Fusion Viewer**: Click any card to see all possible fusion combinations
- **Search Functionality**: Quickly find cards by name or description
- **Responsive Design**: Works seamlessly on desktop and mobile devices
- **Real Card Images**: Card artwork provided by [YGOProDeck API](https://ygoprodeck.com/api-guide/)

## 💡 How to Use

1. **Browse Cards**: Scroll through the card gallery to see all available cards
2. **Search**: Use the search bar to find specific cards by name or description
3. **View Fusions**: Click on any card image to open a modal showing all possible fusions
4. **Fusion Formula**: See the exact combination (Card 1 + Card 2 = Result) for each fusion
